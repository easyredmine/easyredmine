# Easy8 (formerly Easy Redmine)

**No. 1 WorkOps platform where people, tools, and AI connect into one ecosystem of work.**

Easy8 is the only EU-compliant software that breaks the barriers between your teams. Gain clarity & control across all your projects, support and software development.

![Easy8 Product Overview](https://www.easy8.com/EasyRedmine/media/site-images/easy-redmine-product-oveview.png)

## Key features

Easy8's comprehensive capabilities include:

- [Gantt Charts](https://www.easy8.com/redmine-gantt-plugin) - a visual project management tool that displays a project's schedule, tasks, and dependencies over a timeline
- [Resource Management](https://www.easy8.com/product/resource-management) - allocate resources effectively across multiple projects
- [Risk Management](https://www.easy8.com/product/risk-management) - identify potential issues before they impact your projects
- [Helpdesk and IT service management](https://www.easy8.com/product/it-service-management) - manage customer support and maintain strict SLAs
- [Reporting](https://www.easy8.com/product/time-tracking-reporting) - gain visibility into project status, resource utilisation and financials
- [Knowledge Base](https://www.easy8.com/product/knowledge-base) - centralise information with an integrated wiki system (powered by Wikipedia engine)
- [B2B CRM](https://www.easy8.com/product/sales-crm) - manage B2B client relationships throughout the project lifecycle
- [WBS](https://www.easy8.com/redmine-wbs-plugin) - Work Breakdown Structure for detailed project planning and organization
- [Agile project management](https://www.easy8.com/product/agile-tools) - implement agile methodologies with specialized tools and workflows (Scrum Boards, Kanban Boards)
- [Time tracking](https://www.easy8.com/product/time-tracking-reporting) - track time spent on tasks and generate detailed reports
- [GitLab &amp; GitHub integration](https://www.easy8.com/product/source-code-management) - seamlessly integrate with popular source code repositories
- [Easy AI](https://www.easy8.com/product/easy-ai) - the first on-premises AI for project management, deliver projects and tasks efficently without sacrificing security
- [Automations &amp; AI agents (powered by n8n)](https://www.easy8.com/product/automations-ai-agents) - automating routine tasks, connecting tools, and keeping projects in motion so your people can focus on what truly matters.

## Open source plugins

As an open source available software, we offer several free plugins to enhance your Redmine experience:

- [Redmine Gantt Chart plugin](https://www.easy8.com/redmine-gantt-plugin) - Visualise project timelines and dependencies
- [Redmine Work Breakdown Structure (WBS) plugin](https://www.easy8.com/redmine-wbs-plugin) - Create hierarchical project structures for better planning and execution
  Our plugins are compatible with standard Redmine installations and can be used independently of the full Easy8 WorkOps Platform.
